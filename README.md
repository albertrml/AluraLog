# AluraLog — Estudo de Caso em Business Intelligence (Projeto Evolutivo)

## Visão Geral
O **AluraLog** é um projeto de *Business Intelligence* desenvolvido como **estudo de caso de análise logística**, com foco em **modelagem de dados, definição de KPIs e geração de insights para tomada de decisão**.

Este repositório foi concebido como um **projeto evolutivo**, no qual o **mesmo problema de negócio** é progressivamente resolvido utilizando **diferentes ferramentas e tecnologias**, refletindo uma trajetória realista de amadurecimento profissional na área de dados.

> **Projeto em evolução contínua**  
> As análises aqui apresentadas são reimplementadas conforme novas ferramentas são incorporadas ao stack (Excel → Power BI → SQL → Python), mantendo consistência de modelo e indicadores.

---

## Objetivo do Projeto
Simular um cenário real de análise de dados logísticos, permitindo:
- Monitorar indicadores operacionais (SLA, entregas no prazo, atrasos)
- Avaliar desempenho por região e unidade federativa
- Criar uma base analítica estruturada para BI
- Evoluir a análise sem alterar o problema de negócio

## KPIs e Indicadores Analisados
- Total de entregas realizadas
- Entregas dentro e fora do prazo
- Percentual de SLA atendido
- Tempo médio de entrega
- Análise por região e estado (UF)

Os indicadores foram definidos com base em **necessidades reais de gestão logística**, priorizando clareza e aplicabilidade gerencial.

---

## Metodologia
O projeto segue boas práticas de BI e análise de dados:

- Modelagem de dados analíticos (tabelas fato e dimensões)
- Processo de ETL estruturado
- Separação entre dados brutos e dados tratados
- Cálculo consistente de métricas
- Organização voltada para escalabilidade e reuso

---

## Fase 1 — Excel BI (Concluída)
**Ferramentas:** Excel, Power Query, Power Pivot

Nesta fase, todo o projeto foi desenvolvido **integralmente em Excel**, contemplando:
- ETL utilizando Power Query
- Modelagem de dados no Power Pivot
- Criação de relacionamentos entre tabelas
- Construção de KPIs logísticos
- Base preparada para futuras integrações com BI

Os arquivos estão disponíveis na pasta `/Excel`.
